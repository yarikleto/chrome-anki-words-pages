# Anki Words Pages

Public static website for Anki Words. The Chrome extension source lives in a separate private repository.

Live: https://yarikleto.github.io/chrome-anki-words-pages/ — published by GitHub Pages from the `main` branch root.

## What the extension does

Items marked **Premium** need a licence; everything else is free.

**Reading**

- Look up a word without leaving the page: double-click, a selection icon, instant translation on selection, `Ctrl+Shift+L`, or right-click → Translate and save. Any of these can be turned off.
- Translation card shows parts of speech, audio via the system voice, and English IPA transcription.
- Bilingual example sentences from Tatoeba; click one to attach it as the card's context.
- Saved words are highlighted when they appear again on a page.
- Highlight unknown words above a CEFR reading level (A2, B1, B2, C1). **Premium**
- Click a word in YouTube captions to pause the video and translate it. **Premium**

**Vocabulary**

- Words are saved with their translation, the sentence they came from, and a link to the page.
- Search, edit, and delete; optionally auto-tag each word with the domain it came from.
- Import from CSV, TSV, or plain text; export to CSV; full JSON backup of settings, vocabulary, progress, and review log.
- Export an Anki `.apkg` deck. **Premium**
- Organise words with tags and decks. **Premium**

**Review**

- Spaced repetition with SM-2 and Anki's four grades: Again, Hard, Good, Easy.
- New words per day, a review cap, card order, a daily goal with a streak, and an optional daily reminder.
- FSRS scheduling with a desired-retention target (0.80–0.95). **Premium**
- Extra review modes — random, reverse, typing, cloze, and audio. **Premium**
- Detailed review statistics. **Premium**

**Languages and storage**

- 109 interface languages, 109 translation languages, and source-language auto-detect.
- Translation runs through Chrome's on-device translator, Google, or MyMemory — automatically, or pinned to one.
- No account or API key. Vocabulary and review progress stay in the browser profile; the free tier holds up to 500 words.

Selected text is sent to a translation service only when you ask for a translation.
